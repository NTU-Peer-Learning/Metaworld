## Installation

```bash
export PYTHONPATH=/path/to/metaworld
```

## Usage

```python
from metaworld.envs import ALL_ENVIRONMENTS as ALL_ENV

print(ALL_ENV)
```

## Dimensions

### MetaWorld

- Observation space: 39
  - current, previous: 18 x 2
    - gripper xyz: 3 (0, 1, 2)
    - gripper distance apart: 1
    - object1 xyz: 3
    - object1 quat: 4
    - object2 xyz: 3
    - object2 quat: 4
  - goal pos: 3
- Action space: 4
  - delta xyz: 3
  - gripper position: 1 (7)
